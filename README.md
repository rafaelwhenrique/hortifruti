# Sistema de Caixa - Hortifruti

Bem-vindo ao Sistema de Caixa do Hortifruti! Este sistema permite que você realize compras de frutas, verduras e folhagens, além de gerenciar formas de pagamento e cadastro de clientes.

## Como Iniciar

### Pré-requisitos

Antes de começar, certifique-se de ter o Python instalado em seu sistema. Você pode baixar a versão mais recente do Python em [python.org](https://www.python.org/downloads/).

### Estrutura do Projeto

O projeto possui a seguinte estrutura de diretórios:

/hortifruti
    ├── caixa.py
    └── produtos.py


### Instalando Dependências

Atualmente, o projeto não possui dependências externas. Para rodar o sistema, basta ter o Python instalado.

## Clonando o Repositório

Para clonar o repositório, siga os passos abaixo:

1. Abra o terminal (CMD, PowerShell, Git Bash, etc.).
2. Navegue até o diretório onde deseja clonar o projeto.
3. Execute o seguinte comando:
   ```cmd
   git clone https://github.com/rafaelwhenrique/hortifruti.git
